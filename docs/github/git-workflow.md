```markdown
# Workflow Git pour contributeurs

## 🎯 Règles simples

```
main     = contenu fini/publié
draft/*  = brouillons (invisibles)
```

## 🚀 Pour tes labs étudiants

1. **Fork** ce repo
2. **Crée ta branche** : `tonnom-lab-dvwa`
3. **Push** tes brouillons → **privé**
4. **Pull Request** vers `main` → review → merge

## Exemple DVWA
```
draft/ctfs/dvwa/brute-force/  → WIP
main/ctfs/dvwa/brute-force/   → publié
```
```
